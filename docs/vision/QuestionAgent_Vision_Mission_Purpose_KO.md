# QuestionAgent – Vision, Mission, Purpose

## 1. Vision (비전)

QuestionAgent는 프로그래밍 지식이 없는 사람도  
자신이 만들고 싶은 프로그램을 자연어만으로 설계하고 실현할 수 있는 환경을 만드는 것을 목표로 한다.

프로그래밍은 복잡하고 전문적이지만,  
“무엇을 만들고 싶은지 설명하는 일”은 누구나 할 수 있다.

QuestionAgent는 사용자가 자신의 아이디어를 언어로 설명하기만 하면  
그 아이디어가 구현 가능한 형태의 프로그램 구조로 재구성되는 과정을 지원한다.

### Vision Statement
누구나 자연어로 프로그램 개발을 시작하고 완성할 수 있는 세계를 만든다.

---

## 2. Mission (역할)

QuestionAgent의 핵심 역할은  
사용자가 말로 표현한 아이디어를 질문을 통해 정제하고 구조화하여  
개발 가능한 요구사항 문서로 변환하는 것이다.

사용자는 기능을 정확히 설명하지 못할 수 있으며,  
중요한 사항을 놓치거나 모순을 만들 수도 있다.  
QuestionAgent는 필요한 질문을 먼저 제시하고,  
표현되지 않은 요소를 추론하여 보완하며,  
결과를 체계적으로 정리된 명세서 형태로 제공한다.

또한 이렇게 구조화된 요구사항을 기반으로  
Codex와 같은 외부 개발 모델이 코드를 생성할 수 있도록  
명확한 개발 지시를 구성하는 역할도 수행한다.

### Mission Statement
사용자의 불명확한 생각을 질문을 통해 구체화하고,  
이를 개발 가능한 형태로 정리하여  
외부 개발 AI가 실제 프로그램으로 구현할 수 있도록 돕는다.

Mission의 핵심 기능:
1. 요구사항을 질문 기반으로 명확하게 파악한다.  
2. 누락·모순을 자동으로 탐지하고 보완한다.  
3. 기능, UX 흐름, 데이터 구조 등을 체계적으로 문서화한다.  
4. 외부 코드 생성 AI가 이해할 수 있는 개발 지시로 변환한다.

---

## 3. Purpose (궁극적 목적)

QuestionAgent의 최종 목적은 다음 한 문장으로 정의된다.

### Purpose Statement
비개발자가 채팅만으로 자신이 원하는 프로그램을 설계하고,  
Codex와의 연동을 통해 실제 실행 가능한 프로그램을 완성할 수 있도록 한다.

이 목적은 다음을 포함한다:

1. 사용자가 만들고 싶은 것을 정확하게 정의하도록 돕는다.  
2. 질문 기반으로 요구사항을 도출하고 구조화한다.  
3. 구조화된 문서를 기반으로 Codex가 코드를 생성할 수 있도록 명확한 지시를 제공한다.  
4. 비개발자도 개발 지식 없이 프로그램을 완성할 수 있는 환경을 제공한다.  
5. 프로젝트 단위 문서·템플릿·기록 관리 체계를 제공하여  
   생성된 프로그램이 확장 가능하고 재현 가능한 형태를 갖추도록 한다.

---

## 핵심 요약

- Vision: 자연어만으로 프로그램 개발을 완성할 수 있는 환경을 만든다.  
- Mission: 사용자의 아이디어를 질문 기반으로 정제하고 개발 가능한 구조로 변환한다.  
- Purpose: 비개발자가 채팅만으로 프로그램을 설계하고 완성할 수 있게 한다.
