# Question List Generation I/O Specification (KO)

이 문서는 QuestionAgent가 LLM을 사용하여
질문 리스트를 생성하기 위한 입력 및 출력 규격을 정의한다.

본 규격은 Web Master Plan Template을 채우기 위한 질문 생성을 목적으로 한다.

---

## 1. 입력(Input)

### 1.1 사용자 정보
- 사용자 기술 수준
  - 비개발자 / 기초 사용자 / 경험자 / 개발자
- 사용 목적 한 줄 요약

### 1.2 프로젝트 정보
- 대상 도메인 (웹, 도구, 기타)
- 적용할 마스터 플랜 템플릿 식별자

### 1.3 제약 조건
- 기술 용어 직접 질문 금지
- 선택지 기반 질문 우선
- 기본값 제안 포함

---

## 2. 출력(Output)

각 질문은 다음 정보를 포함한다.

- 질문 텍스트
- 질문 유형 (선택형 / 자유 입력형)
- 대상 템플릿 필드 경로
- 기본값 또는 추천값

---

## 3. 질문 생성 원칙

- 하나의 질문은 하나의 템플릿 필드만 채운다.
- 질문에는 하나의 의도만 포함한다.
- 사용자가 모를 수 있는 경우 선택지를 제공한다.

---

## 4. 사용 흐름

1. 템플릿 필드 목록에서 질문 대상 추출
2. 사용자 수준에 맞게 질문 난이도 조정
3. 질문 리스트 생성
4. 사용자 응답 수집
5. 템플릿 필드에 응답 매핑
