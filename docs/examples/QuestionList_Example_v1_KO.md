# Question List Example v1 (KO)
## 대상: SystemMgmt_WebTool_Sample_v1

본 문서는 QuestionAgent가 LLM을 통해 생성해야 할 질문 리스트의 기준 예시이다.
실제 서비스용 질문이 아니라, 질문 생성 품질과 템플릿 매핑을 검증하기 위한 레퍼런스 문서이다.

---

## A. 프로젝트 개요

### Q1
- 의도: 프로그램 목적 파악
- 대상 필드: 프로젝트 개요 > 목적
- 질문 타입: 자유 입력
- 질문:
  이 프로그램으로 어떤 일을 관리하고 싶으신가요? 한 문장으로 설명해 주세요.

### Q2
- 의도: 주요 사용자 파악
- 대상 필드: 프로젝트 개요 > 대상 사용자
- 질문 타입: 선택형
- 질문:
  이 프로그램은 누가 주로 사용하나요?
  - 관리자만
  - 관리자와 내부 직원
  - 아직 정해지지 않음

### Q3
- 의도: 사용자 규모 파악
- 대상 필드: 프로젝트 개요 > 사용자 수
- 질문 타입: 선택형
- 질문:
  이 프로그램을 사용하는 사람은 몇 명 정도인가요?
  - 1~5명
  - 6~20명
  - 21명 이상
  - 잘 모르겠음

## C. 핵심 관리 대상 (CRUD)

### Q6
- 의도: 관리 대상 정의
- 대상 필드: 기능 범위 > 관리 대상
- 질문 타입: 자유 입력
- 질문:
  가장 중요하게 관리하고 싶은 정보의 이름은 무엇인가요?

### Q7
- 의도: 관리 대상 설명
- 대상 필드: 기능 범위 > 관리 대상 설명
- 질문 타입: 자유 입력
- 질문:
  이 정보로 어떤 내용을 관리하려고 하나요?

### Q8
- 의도: 필요한 기능 선택
- 대상 필드: 기능 범위 > CRUD 기능
- 질문 타입: 선택형 (복수)
- 질문:
  어떤 기능이 필요할까요?
  - 목록 보기
  - 상세 보기
  - 등록
  - 수정
  - 삭제
